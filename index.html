Explicar concepto y ejemplo de Pseudo elementos y Pseudo Clases

Hover

La propiedad "hover" son una forma efectiva de mejorar la experiencia del usuario al proporcionar retroalimentación visual y destacar elementos interactivos en una página web cuando se pasa el mouse sobre el elemento.

Active

Es un pseudo-clase que se utiliza para seleccionar y estilizar un elemento mientras está siendo activado o siendo clickeado.

Visited

Esta pseudo-clase permite cambiar el estilo de los enlaces después de que un usuario ha hecho clic en ellos y ha visitado la página vinculada.

Input focus y disabled

focus: se aplica a un elemento que está actualmente enfocado, es decir, el elemento que ha recibido la atención del usuario.

disabled: se aplica a elementos que están desactivados. Un elemento puede estar desactivado, por ejemplo, si es un campo de entrada y se ha establecido el atributo disabled.

First child

Es un selector que se utiliza para seleccionar el primer hijo de un elemento padre.

Last child

Es un selector que se utiliza para seleccionar el último hijo de un elemento padre.

Nth:Child

Es un selector en CSS que te permite seleccionar y estilizar elementos específicos en función de su posición en relación con sus hermanos dentro de un contenedor.

After y Before

Son pseudo-elementos en CSS que permiten insertar contenido antes y después del contenido real de un elemento, respectivamente. Estos pseudo-elementos se utilizan para agregar contenido decorativo, iconos, o incluso texto adicional a un elemento sin tener que modificar directamente el contenido HTML.

First-letter y First line

First-letter: Este pseudo-elemento se utiliza para seleccionar y estilizar la primera letra de un elemento, como un párrafo (p) o un encabezado (h1).

First-line: Este pseudo-elemento se utiliza para seleccionar y estilizar la primera línea de un bloque de texto.
